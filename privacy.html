Clacky Privacy Policy

Last updated: May 21, 2026.

Clacky is a Windows tray app that plays local keyboard and mouse sound effects. The app is designed to react to physical input events without learning what you type.

Windows App Data

Clacky listens for physical key and mouse button identifiers so it can play matching sound effects. It does not collect, sell, share, or transmit typed text, passwords, clipboard contents, document names, account profiles, or cross-site identifiers.

Sound packs, volume, tone, favorites, visualizer settings, launch-at-login preference, and optional local usage counts are stored locally in the user's Windows app data folder.

Website Data

The Clacky website may keep one anonymous aggregate count of playground key presses and clicks. It stores the number of interactions, not the words typed into the playground or a personal identifier.

The website may use Cloudflare Web Analytics for privacy-preserving page-view metrics. Clacky does not add cookies, ad pixels, session replay, or fingerprinting.
