Clacky Support

Clacky is a Windows tray app for local keyboard and mouse sound effects. Start with the checks below before reinstalling.

Troubleshooting

No sound playing: open Clacky from the Windows system tray, make sure the main toggle is on, confirm volume is above 0%, choose a sound pack, and check the active Windows output device.

Tray icon hidden: open the Windows notification area overflow and pin Clacky if you want it visible. Relaunching Clacky also restores the tray app.

Typing feels delayed: try a local output device first to rule out Bluetooth latency. Clacky plays sounds on a background audio queue and ignores held-key repeats.

Mouse clicks are silent: turn on Mouse Clicks in the tray flyout or Settings, choose a click tier, and use Preview.

Visualizer not showing: enable Reactive Visualizer and try cursor-follow, top-center, or pinned mode.

Sound pack missing: external packs belong in %LocalAppData%\Clacky\Packs, one folder per pack. Reopen Settings after adding a pack.

Contact

Open a GitHub issue at github.com/Stephanvs/clack/issues with your Windows version, audio device, selected sound pack, and what changed before the issue started.
